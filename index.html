About

A bit about me

I got into programming because I wanted to build things, not just study theory. Maps became my focus after realizing how much untapped potential exists in geographic data, with most of it sitting unused in databases or scattered across government websites.

My approach is practical: find interesting spatial data, clean it up, and build something people can actually use. Whether that's mapping every fire infrastructure data in Sacramento County or visualizing the data center density in Northern Virginia, I like projects where you can point at a map and say "I built that."

Currently finishing my software engineering degree at WGU while working as an intern at ShareVita. Also pursuing a GIS certificate because I believe the best geospatial developers understand both the code and the cartography.

Projects

Things I've built

Sacramento Fire Hydrant Map

I wanted to know where all the fire hydrants in Sacramento were. Turns out, that data doesn't really exist in one place. So I wrote a Python script to query Mapillary's street-level imagery API, which uses computer vision to detect objects in photos. Pulled 35,000+ hydrant locations, then built an interactive map with marker clustering so your browser doesn't crash trying to render them all.

Python Leaflet Mapillary API GeoJSON

Midpoint Restaurant Finder

Built for the classic "where should we meet?" problem. Enter two addresses, and it calculates the geographic midpoint, then finds restaurants nearby. The backend handles caching and rate limiting so I don't burn through Google Maps API credits. Nothing revolutionary, but it's a full-stack app I use myself.

React Node.js Express Google Maps API

NoVA Data Center Map

Northern Virginia hosts one of the largest concentrations of data centers in the world. I collected addresses from various sources, geocoded them with Python, and mapped 230+ facilities. It's a simple visualization, but it makes the scale of cloud infrastructure in one county surprisingly tangible.

Python Leaflet Geocoding GeoJSON

Discord Message Extractor

A CLI tool for archiving Discord conversations. Parses the HTML exports Discord provides, reconstructs threads, and outputs clean formats. Built it because I wanted to preserve some community discussions before they disappeared. Handles date filtering and multiple export formats.

Python HTML Parsing CLI
